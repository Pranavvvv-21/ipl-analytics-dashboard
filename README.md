# 🏏 IPL Analytics Dashboard (2008–2020)

An interactive data analytics dashboard built with Python, Pandas, and Streamlit to explore IPL match data across 13 seasons.

## 🔗 Live Demo
[Click here to view the live app](https://ipl-analytics-dashboard-3i7cqq4pgjhrrefhdtzcln.streamlit.app)

## 📊 Features
- Team performance analysis (wins by team)
- Toss decision impact analysis
- Season-wise match trends
- Top 10 run scorers of all time
- Top 10 wicket takers of all time
- Top venues by matches hosted
- Filter by season using sidebar

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## 📁 Dataset
IPL Complete Dataset 2008–2020 from Kaggle

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
